### Info

TXT Combiner is a program that combines many text files into one. It has useful options for selecting files by the parameters you need.

## Installation

```sh
npm install txt-combiner
```

### Setup
```bash
make setup
```

### Run tests
```bash
make test
```

### Run code coverage
```bash
make test-coverage
```
